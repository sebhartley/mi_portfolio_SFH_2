Mi portafolio

Aquí puedes encontrar algunos de los proyectos más interesantes en los que he trabajado.

Filtrado de Datos Avanzado: Un script que permite a los usuarios gestionar grandes volúmenes de datos filtrando por palabras clave de manera eficiente.

Predicción de Ventas con Machine Learning: Utilicé algoritmos de regresión lineal y árboles de decisión para crear un modelo que predice las ventas de una cadena de tiendas en función de datos históricos y tendencias estacionales.

Dashboard Interactivo para Análisis de Clientes: Creación de un dashboard interactivo en Python (con Dash) para analizar el comportamiento de los clientes, mostrando visualizaciones clave como el valor del cliente a lo largo del tiempo y patrones de compra.

Análisis de Sentimientos en Redes Sociales: Desarrollé un modelo de procesamiento de lenguaje natural (NLP) que analiza las publicaciones en redes sociales para detectar el sentimiento general sobre diferentes productos, utilizando técnicas de clasificación con Naive Bayes y redes neuronales.
